  options={[
    "The stable channel is faster to download",
    "Pinned versions break when their associated flutter_lints package is yanked from pub.dev, while channel: stable always gets the latest compatible release",

<QuizQuestion
  question="What is the most important cross-check for AI-generated release notes?"
  options={[

  explanation="AI-generated release notes sometimes include features that were discussed or planned but never implemented, or overstate the impact of small changes. <QuizQuestion

  options={[

  correctIndex={1}

/>

<QuizQuestion
