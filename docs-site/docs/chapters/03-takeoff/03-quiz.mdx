<QuizQuestion
  question="When is it faster to type code yourself instead of using AI?"
  options={[
    "Never — AI is always faster than typing",
    "Only for comments and documentation",
    "For short, trivial code (1-3 lines) where the AI feedback loop takes longer than typing",
    "Only when the AI tool is offline"
  ]}
  correctIndex={2}
  explanation="The AI feedback loop — writing a prompt, waiting for a response, reading the suggestion, deciding to accept — has overhead. For trivial code like 'const SizedBox(height: 16)' or a simple if/else, your keyboard is faster. AI saves time on 20+ line structural code."
/>

<QuizQuestion
  question="What is the 'Backseat Driver' anti-pattern?"
  options={[
    "Telling the AI exactly what to do with overly detailed prompts",
    "Delegating everything to the AI, including tasks that are faster to do by hand",
    "Using multiple AI tools at the same time",
    "Reviewing AI code too thoroughly before accepting it"
  ]}
  correctIndex={1}
  explanation="The Backseat Driver delegates everything — including trivial one-line widgets and import statements — to the AI. The overhead of the AI feedback loop only pays off when the generated code saves more time than the loop costs. Below 5 lines of straightforward code, typing is usually faster."
/>
